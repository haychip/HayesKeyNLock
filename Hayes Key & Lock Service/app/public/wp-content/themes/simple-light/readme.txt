Do you want to create a portfolio with job descriptions or do you need a stylish blog? This theme is what you need!

== Description ==

Do you like sites with lots of photos? If so, try experimenting with this simple theme. In this topic, your images are the main subject, and text is an addition to your images. The theme is great for telling about your work or illustrating your story. The subject already has a slider with simple settings, and the photos on the main page and in categories are a post attachment of your posts. The theme is easily customizable through a customizer.

== Installation ==

1. Click on the 'Activate' button to use your new theme.
2. Create categories of your posts and posts (recommended to attach a photo to your posts).
3. Select the static main page content (text) which you would like to display on the front page.
4. Navigate to Customize > Theme options to customize the look and other details.
5. Navigate to Appearance > Slider options to create a slider.

== Frequently Asked Questions ==

= How do I add a search form to a site? =

The design of the theme provides for the display of the search form in the main menu area, but you can add a search form for example in the sidebar of the post. For both cases, you just need to connect the corresponding widget to the area you need.

= Does the theme support Internet Explorer 11 and earlier? =

The theme has limited support for IE 11 and does not support earlier versions of IE

= How can I customize the output of content (text) on the main page? =

To do this, you need to create a page in "Pages" or you can use a ready-made page with the content you need. Next, in Settings> Reading, select "A static page (select below)" and select this page.

= What are "blocks" for output on the main page? =

The "block" on the main page is the display area of ​​several posts. There are 3 such blocks on the main page. This is very similar to how entries are displayed in categories or products in online stores. These blocks on the main page, in turn, consist of blocks of records - it looks like bricks. Record blocks have different sizes and this is intended by design.

= How can I change the display style of posts on the main page? =

The theme provides the ability to choose from three types of display posts on the main and in categories: blocks of different heights, larger blocks and small blocks. All this is configured through the customizer in the 'Theme options' section.

= How do I customize the display of posts on the main page? =

The theme is conceived in such a way as to enable you to simultaneously display entries from different categories on the main page. It can be for example blog entries, news and something else. You can easily customize the category of posts, the number of posts and the style of displaying posts on the main page through the customizer in the 'Theme options' section.

= How can I display social network icons under the banner? =

Under the slider there is an area for the 'Social icons sidebar' widget. In the admin panel, you just need to add a widget called 'Light theme icon' to this area. Next, configure this widget.

= The widget 'Light theme icon' uses the 'Fontawesome icon class'. What is it? =

The 'Light theme icon' widget uses the 'Fontawesome' icon to display. This works as follows you need to find a suitable icon on the site https://fontawesome.com. The icon will have an entry similar to "<i class ="fab fa-facebook-f"></i>" (this is the facebook icon) all you need to do is just insert the text "fab fa-facebook-f" in the 'Fontawesome icon class' field, this is what goes in quotes after 'class=' .
